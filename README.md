# Sweetshop-Management
The Sweet Shop Management System is a simple Java-based desktop application designed to help manage a sweet shop's inventory efficiently.
It allows the user to:
Add, view, search, update, and delete sweets.
Maintain user accounts (registration).
Store all data persistently in an SQLite database.
Interact through a Swing-based graphical user interface (UI).

What project can do?
 Add new sweets with name, category, price, and quantity.
 View all sweets in a table view.
 Search sweets by name.
 Update quantity and price of existing sweets.
 Delete sweets by ID.
 User registration module.
 Simple and clean Swing GUI.

 Component	           Technology Used
Programming Language	- Java 17+
Database	            -  SQLite
UI Framework          - Java Swing
Database Connectivity	JDBC (sqlite-jdbc driver)
